Why need to register a hospital?

Our aim is to deliver live bed availability data of all the hospital in the web portal. To archive that we need a universal database for all the hospitals; hence all the hospital authorities must register the hospital, so that we will able to collect the data of the hospitals. We also want to provide a hospital bed booking service to the people so that for a emergency a bed in hospital can easily book in advance, and to implement this features we need to give the access to hospital authorities to admit the booked patients from the web portal and this access will only be given to the registered hospitals; and if more hospitals will register then this bed booking features will be available for the more hospitals means more people will be get the benefits.
We also building a system by that we will collect the data of all the admitted patient in the hospital, and from that data we can analysis the data of the healthcare system like how much patients are admitting in a hospital every day, how many patients are releasing every day, in which area the patients suffering from which disease in most of the case, most of the patients are dying from which disease, how many patients are died in hospitals every day; and from that data we will able to examine the scenario of the healthcare system. So to collect these data from your hospitals we need your hospital's information through registration.

How to register?

  1. Enter all the data of the hospital by fill up the form.
  2. After fill up all the input fields press the "Next" button.
  3. Now you have to provide the bed details of your hospital. By login to the Hospital Profile portal you can also add or remove bed any time.
    After complete the first step we will send you a link to the registered email. Form this link you can also complete this step (second step) within 3 days.
  4. In the final step we will verify your given information and if we find everything is okay in your registration details then we will active your account. After activation of the account you can login to get access.

What registered hospitals can do?

  • View Patients' Data: Hospital authority able to see how many patients are admitted in their hospitals, how many patients the hospital released or referred in past few months
  • View Bed Data: How many beds are available in the hospital and in which beds which patients are admitted.
  • Add or Remove Bed: If more bed included in the hospital, hospital authority can add the data of new included bed in web portal. Or a bed is removed in the hospital then hospital authority can remove the bed's data in the web portal
  • Live Oxygen Data: From the web portal hospital administration can see for how long the supply of oxygen will remain. But to get the features hospital authority need to be establish a IoT set up.
