Before starting First Aid!

First Aid is the very essential thing. A firs aid can save a life. Due to lack of good transport system it is not possible to reach the rescue team or health team at the quickly. It is very common scenario that ambulance reach to the lately. We never should wait for the rescue team. We should start the first aid to save the life of the victim.
But Beware! Your one wrong step can push the victim to death. So it advise to you if you have experience or some kind of knowledge then do first aid; otherwise never try to this. If you have not knowledge then ask others in the accident spot if he/she have knowledge about that. If in the accident spot nobody have any proper knowledge then you can do basic first aid.

You should learn first aid to save a life. From here you can start learning.
